# Social Media Manager Agent 

An autonomous AI-powered social media management system that analyzes past engagement data, generates optimized content, and schedules posts automatically.

##  Features

* Audience insight analysis from historical posts
* AI-generated optimized social media content
* Automated scheduling system
* Real-time console dashboard
* Lightweight and fast prototype architecture

##  How It Works

1. Analyze past post engagement
2. Extract audience trends
3. Generate optimized post suggestions
4. Schedule posts automatically

##  Tech Stack

* Python
* Pandas
* Schedule library
* Flask 
* CSV data storage

##  Project Structure

social-media-agent/
│
├── data/posts.csv
├── ai/insight_engine.py
├── ai/content_generator.py
├── agent/agent_controller.py
├── agent/scheduler.py

##  How to Run

```bash
pip install pandas schedule
python main.py
```

Smart Engagement Predictor:
